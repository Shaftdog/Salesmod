## Architecture

The agent system consists of:

1. **Orchestrator** - Coordinates workflow execution
2. **Context Builder** - Aggregates business data
3. **Planner** - Creates strategic action plans
4. **Executor** - Handles card execution
5. **Tools** - Provides chat interface capabilities
