# Project Summary

**Salesmod** is a production-ready, full-featured appraisal workflow management system with integrated AI agents, CRM, case management, and automated workflows.

## Quick Facts

- **Technology:** Next.js 15, TypeScript, Supabase (PostgreSQL), AI Integration
- **Features:** AI Agents, Chat Interface with RAG, Order Management, CRM, Case Management, Properties, Goals
- **Status:** Production Ready
- **Database:** 25+ tables with Row Level Security
- **Documentation:** 215+ organized documents

## Key Capabilities

1. **AI Agent System** - Orchestrator, context builder, planner, and executor agents
2. **Chat Interface** - RAG-powered conversational UI with intelligent memory
3. **Order Management** - Complete order lifecycle with kanban boards
4. **Case Management** - Structured workflow for appraisal cases
5. **CRM Features** - Client management, contacts, activities, and relationships
6. **Property Management** - Property and unit tracking with addresses
7. **Admin Panel** - User management and system configuration
8. **Goals System** - Track and monitor organizational goals
9. **Email Integration** - Automated email workflows and templates

## Architecture Highlights

- **Frontend:** React 18 with Next.js 15 App Router, Tailwind CSS, shadcn/ui
- **Backend:** Supabase with PostgreSQL, Row Level Security, real-time subscriptions
- **AI:** Vercel AI SDK supporting OpenAI and Anthropic models
- **State:** TanStack React Query for data fetching and caching
- **Forms:** React Hook Form with Zod validation
- **Testing:** Playwright E2E and Vitest unit tests
