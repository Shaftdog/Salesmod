# Archive

This folder contains old work that has been archived for reference but is no longer actively used in the codebase.

## Contents

| Folder | Description |
|--------|-------------|
| `migrations-2025-12/` | Database migrations from December 2025 that were applied directly to production |

## Purpose

Files are archived here when:
- They were applied directly to production but never committed
- They're superseded by newer implementations
- They're useful for historical reference but shouldn't be in the active codebase

Each subfolder contains its own README explaining what's stored and why.
